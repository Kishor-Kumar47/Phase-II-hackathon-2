# Todo Frontend

This is the frontend for the Todo application built with Next.js and Tailwind CSS.

## Features

- Modern, responsive UI with red and black theme
- Public landing page with hero section and feature highlights
- User authentication (sign up and sign in)
- Secure todo management dashboard
- Task creation, viewing, updating, and deletion

## Tech Stack

- Next.js 14.0.3
- React 18.2.0
- Tailwind CSS 3.3.5
- Axios for API calls

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```
